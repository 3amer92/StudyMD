### Download

To download for Mac, click [here](/dist/mac/StudyMD.app).

### Run Locally

You will need yarn and git.

```bash
# Clone this repository
git clone https://github.com/jotron/StudyMD.git
# Go into the repository
cd StudyMD
# Install dependencies
yarn install
# Run the app
yarn start
```
